# MLOps-DSPM

This repository hosts the MLOps-based Data Science process model, including the individual models and accompanying materials. 

## MLOps-DSPM Diagrams

The full process model currently consists of 14 diagrams, with 7 BPMN models detailing the project flow and 7 UML information flow diagrams for the system view, including a particular focus on the MLOps components. 
While 12 diagrams provide full detail of the six Data Science lifecycle stages and can be found in the respective subfolders, the two abstract models (one BPMN and one UML) for the entire project view can be found in the root.

## Additional Material

The repository also contains the design requirements guiding the development and evaluation of the MLOps-DSPM. Furthermore, DSPM_material.pdf provides an additional textual description of the process model. 
Further materials, including application examples, best practices, and refinements will be provided soon.
